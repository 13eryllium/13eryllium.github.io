Lambda ships as a GMS2 extension. Add the `.yymps` to your project, then place `.lam` files in **Included Files** so they're bundled with your game build.

::callout
**VS Code Extension:** Full editor support is available - syntax highlighting, real-time linting for undeclared/unused variables, auto-formatting, and minification.
