::hero
Lambda is a dynamically-typed scripting language designed to run inside **GameMaker Studio 2** games, the same way Lua powers modding in other engines.

::callout-gold
**What Lambda is not:** Lambda does not replace GML. It runs alongside it. Lambda scripts are sandboxed guests that interact only through the functions and constants **you** register.
